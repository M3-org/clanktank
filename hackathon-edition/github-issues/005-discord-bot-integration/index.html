    Discord Bot for Community Voting

    Overview

    Adapt council-bot-enhanced.py to enable community voting on hackathon projects through Discord reactions.

    Background

    We have an existing Discord bot for council voting. We need to modify it for hackathon project voting where community reactions influence final scores (up to 2 bonus points).

    Requirements

    Reaction Mapping

    • ✅ = General approval (+0.5)
    • 💯 = Technical excellence (+0.5)
    • 🤑 = Economic potential (+0.5)
    • 🔥 = Community vibes (+0.5)

    Maximum community bonus: 2.0 points

    Bot Commands

    !hackathon list - Show all projects with current scores
!hackathon view [id] - Display project details
!hackathon vote [id] - Show voting interface with reaction options
!hackathon leaderboard - Display current rankings

    Tasks

    • [ ] Modify bot configuration for hackathon context
    • [ ] Create project listing functionality
    • [ ] Implement reaction tracking per project
    • [ ] Build vote aggregation system
    • [ ] Create embeds for project display
    • [ ] Add cooldown to prevent spam
    • [ ] Implement unique vote tracking (one per user per project)
    • [ ] Create webhook for score updates
    • [ ] Add admin commands for managing votes

    Project Display Embed

    def create_project_embed(project_data, current_votes):
    embed = discord.Embed(
        title=project_data['project_name'],
        description=project_data['description'],
        color=0x8B5CF6  # Purple theme
    )

    embed.add_field(name="Category", value=project_data['category'])
    embed.add_field(name="Team", value=project_data['team_name'])
    embed.add_field(name="GitHub", value=project_data['github_url'])

    # Current community votes
    embed.add_field(
        name="Community Reactions",
        value=f"✅ {current_votes.get('check', 0)} | "
              f"💯 {current_votes.get('hundred', 0)} | "
              f"🤑 {current_votes.get('moneybag', 0)} | "
              f"🔥 {current_votes.get('fire', 0)}"
    )

    return embed

    Vote Tracking

    class HackathonVoteTracker:
    def __init__(self):
        self.votes = {}  # {project_id: {user_id: [reactions]}}

    def add_vote(self, project_id, user_id, reaction):
        if project_id not in self.votes:
            self.votes[project_id] = {}

        if user_id not in self.votes[project_id]:
            self.votes[project_id][user_id] = []

        # Check if user already used this reaction
        if reaction not in self.votes[project_id][user_id]:
            self.votes[project_id][user_id].append(reaction)
            return True
        return False
